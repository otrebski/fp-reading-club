## Micro-workshop
Microworkshop is proposition of workshop for 45-90 minutes. It can be a list of exercises for specific FP topics.

Topics propositions:
- Reader Monad
- Writer Monad
- Monad transformer
- Semigroupal
- Validated
- Foldable
- Traversable
- Property based testing
- IO
